=== How These Connect to the ID Badge App Solution ===
To fully understand the solution code, study these key concepts:
- **MaterialApp & Scaffold**: They form the root structure and the screen's background/app bar.
- **Container & BoxDecoration**: We used this to create the dark grey ID card with rounded corners and a shadow.
- **Column & Row**: Columns stack items vertically (the overall ID card content), Rows stack horizontally (the icon + contact text).
- **Expanded Widget**: We used `Expanded` to prevent the text from overflowing the screen on small devices.
- **CircleAvatar**: Used to create that quick, circular profile picture.
- **Custom Functions**: `buildContactRow` demonstrates how you can avoid repeating code by creating a helper method that returns a Widget.
